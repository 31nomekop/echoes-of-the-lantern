Echoes of the Lantern - Prototype v0.1

What this is:
- A playable offline-ready PWA prototype shell for iPhone or browser use.
- Includes intro, name entry, companion reveal, sanctuary hub, archive starter, discovery popup, autosave cue, and subtle motion.

How to use:
1. Upload the whole folder to a GitHub repo named echoes-of-the-lantern.
2. Enable GitHub Pages from the main branch / root.
3. Open the Pages URL in Safari on iPhone.
4. Open once while online so the service worker caches files.
5. Use Share > Add to Home Screen.
6. After first load, it should work offline.

Important:
- This is the first packaged prototype, not the full finished game yet.
- Progress saves in localStorage on the device/browser.
